    Detailed Description

    This class acts as the main window for the common print dialog. It is a container to widget objects namely tabs, root, preview and controls.

    masterLayout |-tabs |-root -- preview |-controls

    Member Function Documentation

    QCPDialog::QCPDialog(QPrinter * printer, QWidget * parent = Q_NULLPTR)

    Constructs QCPDialog with parent as the parent widget and printer as the printer. The printer handles all the job options that get reflected in the preview as well as the final job output.
