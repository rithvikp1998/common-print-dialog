    Detailed Description

    Preview class creates a widget to display the preview of the print job

    Member Function Documentation

    Preview::Preview(QPrinter * _printer, QString uniqueID, QWidget * parent = Q_NULLPTR)

    Constructs Preview objects with parent as the parent widget and _printer as the printer. The _printer is used further to change the job options that get reflected in the preview. The uniqueID is used a create a pdf file in /tmp/

    Preview::~Preview()

    Destroys Preview objects

    [slot] void Preview::print(QPrinter * printer)

    The function acts as a slot for the paintRequested signal emitted by QPrintPreviewWidget. The function uses poppler and printer settings to generated and paint the image.

    void Preview::resize(const QRect & rect)

    resize() takes a QRect& rect as a parameter and uses it to resize the Preview objects to the same dimensions as the parameter rect

    void Preview::setCollateCopies(bool enabled)

    This function takes a boolean enabled as a parameter. A value of '1' makes the printing in collated order whereas a value of '0' keeps the papers uncollated.

    void Preview::setNumCopies(int copies)

    This function takes an integer copies as a parameter and sets the number of copies of the project to be printed to copies.

    void Preview::setOrientation(const QString & orientation)

    This function takes a string orientation as a parameter and sets the orientation of the paper in the Preview accordingly. orientation can take two values "potrait" and "landscape" which sets the orientation of the preview to potrait and landscape respectively.

    void Preview::setPageSize(QString name, qreal width, qreal height, QString unit)

    This function sets the size of the preview to height height and width width where unit indicates the unit in which the page's size is specified. unit can take two values: "mm" and "in" which indicate that the size is measured in millimeters and inches respectively. If name is null then the standard localized name will be used. If a custom page size then a custom name in the format "Custom (width x height)" will be created.

    [slot] void Preview::setZoom(qreal zoomFactor)

    This function acts as a slot for the zoomSliderValueChanged signal emitted from pages/preview_toolbar.qml and sets the size of the page shown in the preview according to the zoomFactor.

    [slot] void Preview::showNextPage()

    This function acts as a slot for the nextPageButtonClicked signal emitted from pages/preview_toolbar.qml and shows the next page in the document in the preview window.

    [slot] void Preview::showPrevPage()

    This function acts as a slot for the prevPageButtonClicked signal emitted from pages/preview_toolbar.qml and shows the previous page in the document in the preview window.
